Linux rsync命令使用指南

发表于 2018-03-20 | 更新于: 2019-10-26 | 分类于 Linux | | 阅读次数:

1. 概述

rsync 命令是一个远程数据同步工具，可通过 LAN/WAN 快速同步多台主机间的文件。rsync 使用所谓的 “rsync算法” 来使本地和远程两个主机之间的文件达到同步，这个算法只传送两个文件的不同部分，而不是每次都整份传送，因此速度相当快。rsync 是一个功能非常强大的工具，其命令也有很多功能特色选项。
